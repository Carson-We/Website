내 프로젝트

Advanced Retrieval and Inference Engine for Learning(A.R.I.E.L.)

A.R.I.E.L. 은 자연어 처리와 기계 학습을 활용하여 테스트 관련 작업을 지원하며 생산성과 효율성을 향상시킵니다.

Autonomous Neural Avatar（A.N.A.）

A.N.A. 는 개인화된 외모와 행동을 가진 실제같은 디지털 아바타를 생성하여 가상 커뮤니케이션과 자기 표현을 혁신합니다.

A.N.A.는 내년 제2분기에 출시될 예정입니다.

Cantonese Spoken Question-Answer Dataset for Secondary School(CSQADSS)

CSQADSS는 국어 이해와 능숙성 향상을 위해 설계된 데이터셋입니다.

FERD

FERD 는 컴퓨터 비전을 사용하여 얼굴 표정을 분석하여 감정을 추론합니다.

Fruit Classifier

Fruit Classifier 기계 학습과 컴퓨터 비전을 사용하여 과일 이미지를 정확하게 분류합니다.

Lane-Keeping and Obstacle-Avoidance System(L.K.O.A)

L.K.O.A. 은 센서, 카메라 및 기계 학습 알고리즘을 결합하여 안전하고 자율적인 운전을 보완하며 도로 안전과 운전자 지원 기능을 개선합니다.

L.O.K.A. will be launched in the first quarter of next year.

MindCheck

MindCheck 는 SwiftUI로 구축된 iOS 플랫폼으로 종합적인 정신 건강 평가와 지원을 제공합니다.

NeuralMorse

NeuralMorse 는 신경망을 활용하여 텍스트를 모스 부호로 번역하고 그 반대로 번역합니다.

Image Classifier

Image Classifier 는 시각적 특징과 패턴을 분석하여 이미지를 다른 클래스나 범주로 정확하게 분류하는 컴퓨터 비전 시스템입니다.

Project V

Project V 는 AI 알고리즘과 데이터 분석을 활용하여 환경적 도전에 대응하고 지구의 보호와 보존을 위한 혁신적인 솔루션을 개발합니다.

Vision-based Distance and Object Analysis(ViDiAD)

ViDiAD 는 컴퓨터 비전을 사용하여 실시간으로 거리와 객체를 분석하여 동키 카의 안전한 내비게이션과 장애물 회피를 지원합니다.

ViDiAD will be launched in the first quarter of next year.

yujin_ive_lora

yujin_ive_lora 는 LoRA 모델을 사용하여 K-pop 그룹 아이브의 멤버인 유진의 사진을 생성합니다. LoRA 모델은 표준 체크포인트 모델을 수정한 작은 Stable Diffusion 모델입니다.

yujin_ive_lora V2 는 내년 첫 분기에 출시될 예정입니다.
